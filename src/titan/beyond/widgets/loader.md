#### Loader

**Description :**

Le widget pour afficher un chargement.

**Utilisation :**

Utilisation rare (12 fois).
Ce Widget est la plupart du temps déjà dans d'autre widget custom, mais il peut être utilisé pour afficher un attente d'un chargement.

**Paramètres :**

- `color (Color)` _optionel_ : La couleur de l'indicateur de chargement (par défaut noir)

**Exemple :**
_detail_page.dart_

```dart
Loader(
	color: AMAPColorConstants.greenGradient2)
```
